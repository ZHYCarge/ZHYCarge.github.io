魔改Editor.md

发表于 2025-03-09 分类于 日常 阅读次数： Waline： 本文字数： 335 阅读时长 ≈ 1 分钟

前言

由于想要使用Editor.md作为自己博客和日常写作的markdown编写工具（主要是可以实时网页端打开真的很好）

所以编写该教程，记录自己的魔改过程

参考文献

为博客加入一个好用的在线MD编辑工具

魔改阶段

涉及方向

本次魔改主要涉及如下几个方面：

  • 通过使用LocalStorage实现自动保存功能
  • 通过使用HTML5上的File System Access API功能实现对本地文件的打开与保存
  • 实现模板化文档编写，简化编写特定文档的繁琐流程

实现内容

通过对项目目录下的index.html文件进行相关修改，最终实现所需要功能。由于对JS编写着实小白，因此不便描述具体信息，详情代码可参考：https://github.com/ZHYCarge/mdw

尾言

后续进展
